About the Kern
==============

The Kern is an attempt to build a set of web project tools with focus

- Project prototyping
- Design
- Content management
- Delivery


Dependencies
------------

The following node.js modules are required

  express
  connect
  socket.io
  underscore
  backbone
  jquery

To install them, just run:

  npm install jquery backbone underscore express connect socket.io
